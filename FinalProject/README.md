# Cloud-Based Online Student Fee Management System

## Project Description

The Cloud-Based Online Student Fee Management System is a web application developed using Django. It helps educational institutions manage student fee records efficiently. Students can log in to view their fee details and balance, accountants can update fee payments, and administrators can manage the system.

## Features

- Student Registration
- Accountant Registration
- Student Login
- Accountant Login
- Admin Login
- Student Dashboard
- Fee Payment
- Balance Checking
- Fee Update
- Responsive User Interface

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap 5
- SQLite

## Future Enhancements

- AWS EC2 Deployment
- Docker Containerization
- CI/CD using GitHub Actions

- Gunicorn
- AWS RDS Database

## How to Run

```bash
python manage.py migrate
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
